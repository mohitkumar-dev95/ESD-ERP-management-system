# Course Enrollment Mini-Project

## Overview
This mini-project is a part of a larger course management system, focused on implementing the **Course Enrollment** feature. It enables students to select and enroll in a minimum of 4 and a maximum of 6 courses. 

The project is built using **Spring Boot** for the backend and **React** for the frontend.

---

## Features
- View available courses.
- Enroll in courses with validation (4-6 course limit).
- Handle prerequisites and additional constraints during enrollment.
- Secure authentication using JWT tokens.

---

## Tech Stack
- **Frontend**: React, Bootstrap
- **Backend**: Spring Boot, REST APIs.
- **Database**: MySQL
- **Authentication**: JWT Tokens

## Setup Instructions

### Prerequisites
- Node.js and npm installed
- Java Development Kit (JDK) installed.
- Spring Boot setup with Maven.
- A configured database.

### Frontend
1. Navigate to the `esd-mini-react` directory.
2. Install dependencies:  
   ```bash
   npm install
3. Start Frontend:  
   ```bash
   npm start

### Backtend
1. Navigate to the `ESD_MINI` directory.
2. Install dependencies:  
- Open this with Intellij IDEA
3. Run the project.
